# 暮雨笺

融合记事本、待办管理与快速随笔记的桌面应用，支持 Markdown 和 LaTeX 实时渲染。提供简体中文与古风两种文字风格。

荧光与月光缠绕，时光在指尖沉淀

墨痕会呼吸，文本在云端轻盈绽放

待办事项的散落星辰，待你依次拾起

是古典是简素，在你的笔下流转

不必懂得复杂编译，你只需自由书写

毕业快乐。从此山高水长，愿这封「暮雨笺」陪你看代码化作公式，等灵感落成春雨。

见字如面，展信舒颜。

## 功能

### 便签管理
- 新建、搜索（结果高亮）、标签多选筛选、归档、回收站
- 标签下拉选择已有标签，支持全局置顶和标签内独立置顶
- 截止日期倒计时，便签排序（更新时间/创建时间/标题/截止日期）

### 编辑器
- CodeMirror 6 语法高亮、自动换行、查找替换（Ctrl+F/Ctrl+H）
- 快捷键：Ctrl+B 加粗、Ctrl+I 斜体、Ctrl+K 链接
- 插入图片（工具栏选择、粘贴、拖拽）
- 笔记链接 `[[标题]]` 点击跳转
- 代码块语法高亮（16 种语言）

### 待办系统
- 独立待办便签，批量添加任务（每行一个）
- 每任务独立截止时间倒计时
- 每任务正向计时（秒表），记录用时数据
- 任务统计窗口：今日/本周/本月用时饼状图和时间线
- 未完成待办自动置顶，全部完成变灰
- 悬浮窗口查看，支持透明度调节

### 快速笔记
- `Alt+Q` 呼出悬浮速记小窗口
- 关闭时自动保存为便签，支持编辑/预览/分栏模式

### 导出
- 支持 Word、PDF、Markdown、HTML、纯文本格式
- PDF 导出需系统安装 LaTeX 发行版（如 MiKTeX）

### 界面
- 浅色/深色主题切换
- 简体中文/古风文字切换
- 预览面板显示/隐藏（Ctrl+Shift+P）
- 专注模式（F11），沉浸写作

### LaTeX 公式
- KaTeX 实时渲染，支持 `$...$` 行内和 `$$...$$` 块级公式
- 支持 equation/align/gather 等数学环境

### 数据安全
- 回收站：删除的便签可恢复，30 天自动清理
- 自动保存：每 60 秒自动保存，退出时立即保存
- 任务计时数据持久化到本地文件

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式启动
双击「启动暮雨笺.bat」

# 或手动构建并启动
npm run build
npx electron .
```

## 打包为安装程序

```bash
双击「打包.bat」
```

安装包输出在 `release` 文件夹中。

> 注意：pandoc.exe 未包含在 git 仓库中（超过 GitHub 文件限制），打包前请确保 `resources/pandoc/pandoc.exe` 存在。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+Q` | 快速笔记小窗口 |
| `Ctrl+N` | 新建便签 |
| `Ctrl+B` | 加粗 |
| `Ctrl+I` | 斜体 |
| `Ctrl+K` | 插入链接 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 替换 |
| `Ctrl+Shift+P` | 显示/隐藏预览面板 |
| `F11` | 专注模式 |
| `Esc` | 退出专注模式 |
