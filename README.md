# 花笺

融合记事本、时间计划与快速随笔记的桌面应用，支持 Markdown 和 LaTeX 实时渲染。提供简体中文与古风两种文字风格。

## 功能

- **便签管理**：搜索、标签过滤、归档、截止日期倒计时
- **Markdown 编辑器**：CodeMirror 6 语法高亮 + 实时预览
- **LaTeX 公式**：KaTeX 渲染，支持 `$...$` 和 `$$...$$`
- **任务列表**：预览区可点击勾选，自动同步源文本
- **今日计划**：聚合所有标记的便签中未完成任务
- **快速笔记**：`Ctrl+Shift+N` 唤出悬浮小窗口
- **自动保存**：每 60 秒自动保存，退出时立即保存
- **双语风格**：简体中文 / 古风文字一键切换
- **深色模式**：浅色/深色主题，偏好自动保存

## 快速开始

```bash
# 安装依赖（已配置镜像加速）
npm install

# 构建
npm run build

# 启动
双击「启动花笺.bat」
```

## 打包为安装程序

```bash
双击「打包安装程序.bat」
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+N` | 快速笔记小窗口 |
| `Ctrl+B` | 加粗 |
| `Ctrl+I` | 斜体 |
| `Ctrl+K` | 插入链接 |

## 技术栈

Electron + React + TypeScript + Tailwind CSS + CodeMirror 6 + KaTeX + Zustand
